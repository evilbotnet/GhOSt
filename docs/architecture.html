The trick

The shell is a chromeless --app Chromium window pinned as the desktop. Anything a web page can't do — files, ptys, Wi-Fi, native windows, AI — goes through ghostd's authenticated localhost API.

That authenticated bridge is what makes it an operating system, not a browser. Click any node, or trace a flow above.
